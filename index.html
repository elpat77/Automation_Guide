Website Overview

This website was built as a place to keep information on how to to run automation using different programing languages.

It contains things such as installation, tools needed and scripts to use in order to run tests.

Ruby Overview

This guide was created in order to use Selenium Webdriver with Ruby using Mac OS.

There are several methods to install Ruby, including package managers, which are the system-provided tools that manage software packages; installers, which install specific versions of Ruby; managers, which help switch between multiple Ruby installations, and from source code.
Using third-party tools like managers or package managers might be a better idea that allow you to more easily get set up with Ruby.

I recommend using the manager RVM, which stands for the Ruby Version Manager, it's widely used, and allows you to quickly and easily install Ruby.

Link to Ruby RVM Website

Ruby Installation

  • For installing RVM with default Ruby and Rails in one command, type:
    \curl -sSL https://get.rvm.io | bash -s stable --rails


  • If you only want the latest stable version of RVM; just type:
    \curl -sSL https://get.rvm.io | bash -s stable


  • Once it completes, you can confirm that Ruby was successfully installed by typing:
    ruby -v

    • Selenium Webriver for Ruby

    In order to create scripts that interact with the Selenium Server (Remote WebDriver) or create local Selenium WebDriver scripts, you need to make use of language-specific client drivers.

  • Selenium-Webdriver for Ruby can be downloaded on the following link:

    Ruby Selenium

  • By selecting the above link, user will be taken to the Ruby Gems Selenium Webdriver page:

    Ruby Gems

    • Just a note on Gems and Ruby, Gems are independent software packages. You can think of it as a library or plug-in. RubyGems is the package management framework. And each gem needs to be installed before using it. Gems can either be installed via gem install or via the bundler.

  • Install gems by locating the "Gemfile" command and copyting it, it looks something like this:

    gem 'selenium-webdriver', '~> 3.142', '>= 3.142.7'

  • Within your project, create a executable file and open it using a text editor such as Sublime Text. Name this file :Gemfile.
    Once there, the first line of this file should be the source. The source is the webpage, rubygems.org and tells the bundler where to go to search for and install the specified Gems.
    (this is the gemfile source we just copied)

  • In this case the Gemile should contain the following two lines:
    source 'https://rubygems.org'
    gem 'selenium-webdriver', '~> 3.142', '>= 3.142.7'

  • To install and manage Gem dependencies in this file, we must use the bundler. Navigate to the project where the gemfile is located, with terminal and run:
    • gem install bundler
    bundle install
